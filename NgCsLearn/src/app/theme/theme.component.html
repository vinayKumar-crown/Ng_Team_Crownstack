Themes Concept

Themes are the most important part in Web Applications. Through this we can provide different colors, fonts and sizes on every HTML tag. This also help you to utlize your CSS code in proper place.

What are the requirements for Theme creation?

  • Design Guidelines
  • CSS/SCSS Variable Creation

    Variable creation is the most important part of Theme. You have to choose wisely which variables can participate in Theme creation. Now we have two types of variables in Angular:

    1. CSS variables. These we can define in the root of project. It works only globally. You can use it like this. 

                                {{':root {
                                --varName: value;
                                --varName1: value;
                            }
                            .className {
                                color: var(--varName);
                            }'}}
                        
                                {{'document.documentElement.style.setProperty(\'--varName\', \'value\')'}}
                        
                                {{':host {
                                --varName: value;
                                --varName1: value;
                            }
                            .className {
                                color: var(--varName);
                            }'}}
                        
                                {{'this.elementRef.nativeElement.style.setProperty(\'--varName\', \'value\')'}}

    2. SCSS variables. These we can define anywhere in project. You can use it like this. There values can not be changed through Javascript. 

                                $varName: value;
                            {{
                                '.className {
                                    color: $varName
                                }'
                            }}

How to manage theme?

  • From Scratch

    If you are not using any online built theme and thinking for preparing it from start, then you must get the design guidelines. Because these guidelines will help you to add each style property individually. You can manage theme from start like this:

    1. Create theme folder in root of project.

    2. Create variables.scss file. Here you will add all variables.

    3. Now get understanding from design guidelines and create individual files according to those.

    4. Here I have created these files: variables, button, input, accordian, table, anchor etc.
