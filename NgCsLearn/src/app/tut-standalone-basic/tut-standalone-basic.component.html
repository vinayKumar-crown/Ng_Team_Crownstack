Basics of Standalone Components

Standlone Component can not be added in NgModule Declarations

        {{"@NgModule({
            declarations: [
              StandaloneComponentName // This can not be added in declaration. Angular compiler will give error.
            ],
            imports: [
              CommonModule
            ]
          })"}}

You can add Standalone Component in imports of NgModule

        {{"@NgModule({
            declarations: [],
            imports: [
              CommonModule,
              StandaloneComponentName // This can be added in imports.
            ]
          })"}}

You can add another Standalone Component and Modular Dependencies in Standalone Component like below:

        {{"@Component({
            selector: 'app-standalone',
            standalone: true,
            imports: [CommonModule, ReactiveFormsModule, StandAloneComponentB],
            templateUrl: './standalone.component.html',
            styleUrls: ['./standalone.component.css']
          })
          export class StandAloneComponentA implements OnInit {...}"}}
