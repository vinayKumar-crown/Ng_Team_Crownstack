Standalone Components

What we can create as Standalone in Angular?

  • Components
  • Pipes
  • Directives

Angular CLI commands to create the Standalone?

  • ng g pipe pipeName --standalone
  • ng g directive directiveName --standalone
  • ng g component componentName --standalone

Use Standalone Components with?

  • Module-based components
  • Other standalone components
  • Loading routes
  • Lazy loading
